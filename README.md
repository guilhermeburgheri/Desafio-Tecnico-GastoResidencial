# Controle de Gastos Residenciais

Sistema para cadastro de transações, pessoas e consulta de totais.

## Tecnologias

- ASP.NET Core com C#
- Entity Framework Core
- SQLite
- React
- TypeScript
- Vite

## Funcionalidades

- Criação, exclusão e listagem de pessoas
- Exclusão automática das transações de uma pessoa removida
- Criação e listagem de transações
- Bloqueio de receitas para pessoas menores de 18 anos
- Consulta de receitas, despesas e saldo por pessoa
- Consulta dos totais gerais
